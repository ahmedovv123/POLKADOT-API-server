# polkadot-explorer

## Setup with NPM
``` bash
# clone repo
git clone https://github.com/ahmedovv123/polkadot-explorer.git
cd polkadot-explorer

# install dependencies
npm install

# run
npm run server
```
## Setup with Docker
``` bash
# clone repo
git clone https://github.com/ahmedovv123/polkadot-explorer.git
cd polkadot-explorer

# run containers
docker-compose up
```
